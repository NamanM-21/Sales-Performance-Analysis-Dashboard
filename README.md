# Sales-Performance-Analysis-Dashboard
Developed an interactive Sales Performance Dashboard using Power BI to analyze sales trends, customer behavior, product performance, and regional target achievement.
---

## Features
- Sales Overview Dashboard
- Product Performance Analysis
- Customer Insights
- Regional Performance Tracking
- Interactive Filters and Slicers
- Dynamic KPI Reporting

---

## KPIs
- Total Revenue
- Total Customers
- Revenue per Customer
- Product Contribution %
- Target Achievement %
- Loyalty Segmentation

---

## Tools & Technologies
- Power BI
- DAX
- Power Query
- Data Modeling

---

## Dashboard Pages
1. Home Page
2. Sales Overview
3. Product Performance
4. Customer Insights
5. Regional Review

---

## Key Insights
- West region achieved highest target performance.
- High loyalty customers contributed major revenue share.
- Electronics category generated maximum sales revenue.
- Seasonal sales spikes observed in Q4.
